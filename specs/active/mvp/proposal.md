# Proposal — VYVY Text Investigation MVP

## Problem

Users need a fast, understandable way to evaluate suspicious messages without manually searching and reconciling many sources.

## Proposed solution

A text-only web application that performs:

- Fast safety check.
- Structured intake.
- Multi-source evidence collection.
- Parallel expert analysis.
- Behavioral manipulation analysis.
- Judge aggregation.
- Deterministic risk and confidence scoring.
- Explainable report and safety recommendations.

## Why this scope

A one-day build requires a demonstrable vertical slice. Text-only input removes OCR, upload and document-processing risk while preserving the core innovation: investigation, debate, verification and explanation.

## Success measures

- End-to-end demo works.
- Report is evidence-grounded.
- Risk and confidence are separated.
- Provider failure is visible and graceful.
- User receives an immediate safety action.
